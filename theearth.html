TheEarth 1.2

This object comprises a simple scene file and a choice of 3 copyright free images of the earth from https://www.evl.uic.edu/pape/data/Earth/. The scene file maps an image onto a sphere and optionally adds clouds. https://www.evl.uic.edu/pape/data/Earth/ also contains higher resolution images than those incorporated in this object and various other earth images derived from different data sources. All of the images there are in a cylindrical (latitude/longitude) projection, and are suitable for uv mapping to a sphere.

Control Variables

The following variables are set at the top of the sample scene file to facilitate manipulation of the scene:

TheEarth_CloudsOn

Where 0=No and 1=Yes.

TheEarth_Rotation

A 3D <x,y,z> vector in degrees.

TheEarth_SourceImage

The 3 images included in the download are:
theearth_bigearth.jpg ‘True’ colour representation - high resolution
theearth_topography.jpg Colours indicate Heights and Depths
theearth_mapperwdb.jpg World DataBase Country markings ca. 1990
